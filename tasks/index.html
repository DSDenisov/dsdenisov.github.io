What skills do I need to develop for a career in
Business Intelligence and Data warehousing

  6. OLAP tools and understand multidimensional data analysis
  7. Learn about Temporal Tables
  8. Learn about Geo-Spatial data
  9. Learn an ETL tool - Informatica, Pentaho etc.
  10. Learn about OLAP
  11. Learn a reporting tool - Microstrategy, OBIEE etc
  12. Learn Perl/Python/Linux scripting
  13. Learn Performance tuning
  14. Learn to understand the Business Processes/Requirements
  15. Learn Regular Expressions
